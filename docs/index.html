How this works & caveats

Neighborhood shading: U.S. Census ACS table B05006 (place of birth for the foreign-born) at the PUMA level — the 55 named NYC neighborhoods the Census reports country-of-birth for. Spots: NYC DOHMH restaurant inspections (cuisine + location). Spots are not quality-ranked — no free, terms-compliant source lets us re-host restaurant ratings, so each spot links out to Google Maps for live reviews.

This is a heuristic, not a screening guide. It shows where a nation's New Yorkers live and where that cuisine clusters — not which bars actually show the match. Cuisine tags are broad; place of birth isn't ancestry.
